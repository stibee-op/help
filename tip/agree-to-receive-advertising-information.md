# 광고성 정보 수신 동의 여부 정기적으로 확인하기

## 언제 필요한가요?

* 광고성 정보 수신 동의를 정기적으로 확인해야 할 때

***

{% hint style="info" %}
이 가이드에서는 한국인터넷진흥원의 가이드에 따라 구독자의 수신 동의일을 기준으로, 매년 자동으로 수신 동의 여부를 확인하는 방법에 대해 안내하고 있습니다.
{% endhint %}

정보통신망법에서는 광고성 정보 수신자의 ‘수신 동의 여부’를 주기적으로 확인해야 한다고 규정하고 있습니다. 한국인터넷진흥원(KISA)에서 제공하는 '[불법스팸 방지를 위한 정보통신망법 안내서](https://www.kisa.or.kr/401/form?postSeq=3256)'에 따르면, 수신 동의 안내는 수신 동의를 받은 날로부터 2년 이내에 진행하면 된다고 안내합니다. (안내서 37쪽 참고)

자동 이메일의 \[기념일] 트리거를 사용하면, 구독자의 '수신 동의일'을 기준으로 구독자마다 다른 날짜에 자동으로 이메일을 발송할 수 있습니다. 또는 일반 이메일을 활용해 전체 구독자에게 한 번에 발송하는 방식도 가능합니다.



## 자동 이메일 활용하기

스티비의 \[기념일] 트리거를 활용해 매년 자동으로 수신 동의 여부 확인 이메일을 보낼 수 있습니다. 구독자별로 다른 날짜에 개별 발송되므로, 수신 동의 확인을 매년 자동으로 운영하기에 적합한 방식입니다.

### 스티비로 구독자를 모집한 경우

스티비를 통해 구독 신청을 받았다면 \[구독일] 정보를 활용할 수 있습니다. 구독일을 기념일로 설정하면, 해당 날짜를 기준으로 매년 자동으로 수신 동의 여부 확인 이메일이 발송됩니다.

1. \[이메일 → 새로 만들기 → 자동 이메일]을 클릭해 새로운 자동 이메일을 생성합니다.
2. \[주소록] 단계에서 수신 동의를 확인할 구독자가 있는 주소록을 선택합니다.
3. \[발송 정보] 단계에서 \[기념일] 트리거를 선택합니다.
4. 날짜 필드를 \[구독일]로 선택하고, 이 날짜를 기준으로 이메일이 발송될 시점을 설정합니다. (예. 구독일 1일 전, 7일 전 등)
5. \[필터]에서 새로운 필터를 추가한 뒤, \[광고성 정보 수신 동의 → 동의]를 설정합니다.
6. \[콘텐츠]에서 이메일의 내용을 작성합니다.
   * 이때 [메일머지](../email/edit/personalized-merge.md)($%subscribed\_date%$)를 사용하면 각 구독자의 구독일을 개인화해 안내할 수 있습니다.

이렇게 설정하면 처음 구독 신청한 날을 기준으로 '광고성 정보 수신'에 '동의'한 구독자에게 매년 자동으로 확인 이메일이 발송됩니다.

### 외부에서 수신 동의를 받은 경우

외부에서 구독자를 수집했다면 날짜 필드를 하나 생성한 뒤, 이 필드를 기준으로 수신 동의 여부 확인 이메일이 매년 자동 발송되도록 설정할 수 있습니다.

1. 주소록의 \[사용자 정의 필드]에서 ‘수신 동의일’을 입력할 [날짜 유형 필드](https://help.stibee.com/list/adding-managing-subscriber/understanding-subscriber-info#undefined-5)를 생성합니다.
2. 구독자를 주소록에 추가합니다.
   * 1번에서 만든 \[사용자 정의 필드]에 구독자별 수신 동의일을 입력합니다.
   * 수신 동의 상태는 ['동의함'으로 추가](https://help.stibee.com/list/adding-managing-subscriber/add#undefined)합니다.
   * 날짜 형식은 업로드 [형식에 맞게](https://help.stibee.com/trouble-shooting/subscriber-upload-issues#h_01hehqt3qngp7fbhy4zfm55196) 입력합니다.
3. \[이메일 → 새로 만들기 → 자동 이메일]을 클릭해 새로운 자동 이메일을 생성합니다.
4. \[주소록] 단계에서 2번에서 구독자를 추가한 주소록을 선택합니다.
5. \[발송 정보] 단계에서 \[기념일] 트리거를 선택합니다.
6. 날짜 필드를 1번에서 생성한 \[사용자 정의 필드]로 선택하고, 이 날짜를 기준으로 이메일이 발송될 시점을 설정합니다. (예. 구독일 1일 전, 7일 전 등)
7. \[필터]에서 새로운 필터를 추가한 뒤, \[광고성 정보 수신 동의 → 동의]를 설정합니다.
8. \[콘텐츠]에서 이메일의 내용을 작성합니다.
   * 이때 [메일머지](../email/edit/personalized-merge.md)($%subscribed\_date%$)를 사용하면 각 구독자의 구독일을 개인화해 안내할 수 있습니다.

이렇게 설정하면 '수신 동의 날짜 필드' 기준으로 '광고성 정보 수신'에 '동의'한 구독자에게 매년 자동으로 확인 이메일이 발송됩니다.

## 일반 이메일 활용하기

일반 이메일을 활용해 1년에 한 번씩 수신 동의 여부를 확인할 수 있습니다. ‘광고성 정보 수신 동의’가 ‘동의함’인 구독자를 분류한 뒤, 수신 동의 여부 확인 이메일을 보내면 됩니다.

1. \[주소록 → 세그먼트 → 새로 만들기]를 클릭해 새로운 세그먼트를 생성합니다.
2. 조건을 다음과 같이 설정합니다. \[템플릿 불러오기]에서 조건을 가져올 수도 있습니다.
   * 구독자 정보 → 광고성 정보 수신 동의 → 동의
3. \[만들기]를 클릭해 세그먼트를 저장합니다.
4. \[이메일 → 새로 만들기 → 일반 이메일]을 클릭해 새로운 일반 이메일을 생성합니다.
5. \[주소록] 단계에서 3번에서 만든 세그먼트를 선택합니다.
6. \[콘텐츠]에서 이메일의 내용을 작성합니다.
   * 이때 [메일머지](../email/edit/personalized-merge.md)($%subscribed\_date%$)를 사용하면 각 구독자의 구독일을 개인화해 안내할 수 있습니다.

세그먼트와 일반 이메일 기능을 사용해 '광고성 정보 수신'에 '동의'한 구독자를 분류하고 이메일을 보낼 수 있습니다. 1년에 한 번씩 수신에 동의한 전체 구독자에게 이메일을 보내 수신 동의 여부를 확인할 수 있습니다.

_\* '광고성 정보 수신 동의' 여부는 스티비 구독 폼 또는 페이지를 통해_ [_구독 신청 시 받을 수 있고_](https://help.stibee.com/list/adding-managing-subscriber/understanding-subscriber-info#setting-consent-options)_, 주소록에 구독자를 직접추가할 때_ [_동의 여부를 업데이트_](https://help.stibee.com/list/adding-managing-subscriber/add#undefined)_할 수도 있습니다._
