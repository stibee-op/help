# 구독 받기

## 이 글에서는 <a href="#h_01hfna4p2h02gnxargrck6ft11" id="h_01hfna4p2h02gnxargrck6ft11"></a>

구독 폼을 사용해 구독 신청을 받는 방법과, 구독 폼을 설정, 수정하는 방법을 안내합니다. 구독 폼 설정부터 구독 확인 이메일, 외부 사이트에 구독 폼을 설치하는 방법까지 순서대로 살펴봅니다.

***

## 구독 폼 이해하기 <a href="#h_01ggcbmn0c7yk62feh8bwj7xg6" id="h_01ggcbmn0c7yk62feh8bwj7xg6"></a>

구독 폼은 새로운 구독자로부터 이메일 구독 신청을 받을 수 있는 입력 폼입니다.

구독 폼은 주소록과 연결되어 있으며, 주소록을 생성하면 해당 주소록에 연결된 구독 폼이 자동으로 함께 생성됩니다. 구독자는 구독 폼에 이메일 주소를 입력해 직접 구독을 신청할 수 있으며, 구독 폼을 통해 신청한 구독자는 구독 폼과 연결된 주소록에 자동으로 등록됩니다.

구독 폼 예시는 아래에서 확인할 수 있습니다.

* <스티비 뉴스레터>, [https://page.stibee.com/subscriptions/3](https://page.stibee.com/subscriptions/3)
* <위클리D>, [https://page.stibee.com/subscriptions/29178](https://page.stibee.com/subscriptions/29178)
* <커피팟>, [https://page.stibee.com/subscriptions/52057](https://page.stibee.com/subscriptions/52057)



추가된 구독자는 \[주소록 → 구독자 목록]에서 확인할 수 있으며, 구독자는 구독 완료 화면에서 구독한 날짜와 시간을 직접 확인할 수 있습니다.

## 구독 폼 설정하기 <a href="#h_01ggcbmvd2pm93jfz25n2z1kc2" id="h_01ggcbmvd2pm93jfz25n2z1kc2"></a>

구독 폼 설정은 \[주소록 → 구독 관리 화면]에서 진행할 수 있습니다.

1. \[주소록]을 클릭합니다.
2. 구독 신청을 받을 주소록을 선택합니다.
3. 주소록 메뉴에서 \[구독 관리 화면]을 클릭합니다.
4. 화면 상단의 \[구독 폼] 섹션을 확인합니다.

구독 폼 섹션에서는 구독 폼 활성화 여부, URL 확인, 구독 폼 수정, 코드 내보내기 등 구독 폼 사용에 필요한 기본 설정을 관리할 수 있습니다.

## 언어 선택하기

구독 폼, 구독 확인 이메일, 구독 정보 변경 화면에서 사용할 언어를 선택할 수 있습니다. 기본 언어는 한국어이며, 필요 시 영어로 변경할 수 있습니다.

{% hint style="info" %}
\***주의:** 언어를 영어로 변경해도 구독 폼의 제목, 버튼 이름, 구독자에게 노출되는 구독 폼 항목은 직접 수정해 주셔야 합니다. 예를 들어, 구독 폼의 언어를 '영어'로 변경하더라도 '이메일 주소'가 'Email Address'로 자동 변환되지는 않습니다.

필요한 항목은 \[구독 폼 → 수정하기]에서 직접 수정해 주세요.
{% endhint %}

## 구독 폼 활성화 설정 <a href="#id-01hh1t1m2chswbxwsz6rvfh143" id="id-01hh1t1m2chswbxwsz6rvfh143"></a>

구독 폼 활성화 여부를 설정할 수 있습니다.

* 활성화: 구독 폼 URL을 통해 구독 신청 가능
* 비활성화: URL로 접속해도 구독 폼이 표시되지 않음

![](<../../.gitbook/assets/3 (11).png>)

구독 폼 페이지로 연결되는 URL을 확인할 수 있습니다. URL은 자동으로 생성되며 수정할 수 없습니다.

## 구독 폼 정보 수정하기 <a href="#h_01ggcbne0xarz8tagb4jctaedf" id="h_01ggcbne0xarz8tagb4jctaedf"></a>

\[수정하기]를 클릭하면 구독 폼에 표시되는 정보를 수정할 수 있습니다. 설정 가능한 항목은 다음과 같습니다.

* 헤더 이미지
  * 구독 폼 상단에 표시되는 이미지입니다.&#x20;
  * 권장 사이즈는 가로 550px 이상이며, 가로 550px보다 작으면 왼쪽으로 정렬됩니다.
* 제목
  * 구독 폼 상단에 표시되는 제목입니다.&#x20;
  * '주소록 이름+구독하기'가 기본값으로 표시됩니다.
* 내용
  * 구독 폼 제목 아래 표시되는 상세 내용입니다.
* 구독자 입력 정보 설정
  * 구독 신청할 때 입력받을 정보를 설정합니다.&#x20;
  * 이메일 주소는 필수이며, 이름 등 추가 항목은 선택입니다. 구독자 입력 정보 순서는 자유롭게 변경할 수 있습니다.
  * 추가로 입력받고 싶은 정보가 있다면, 사용자 정의 필드에서 먼저 추가하면 됩니다. 사용 방법은 [구독자 정보 설정하기: 사용자 정의 필드](https://help.stibee.com/hc/ko/articles/5659525285007)를 참고해 주세요.&#x20;
* 개인정보 수집 및 이용 동의 표시
  * 체크박스 형태로 표시합니다.&#x20;
  * 표시되는 내용을 편집하고 텍스트 스타일을 설정할 수 있습니다. 내용을 입력하지 않으면 기본값이 표시됩니다.
* 광고성 정보 수신 동의 표시
  * 체크박스 형태로 표시되며, 필수 또는 선택 여부를 설정할 수 있습니다.
  * 표시되는 내용을 편집하고 텍스트 스타일을 설정할 수 있습니다. 내용을 입력하지 않으면 기본값이 표시됩니다.
* 자동 입력 방지(reCAPTCHA) 표시
  * 스팸 봇의 비정상적인 구독 요청을 방지할 수 있습니다.
  * \[코드 내보내기] 방식에서는 표시되지 않습니다.
* 버튼
  * 입력한 정보를 제출하는 버튼입니다.
  * 이 버튼은 구독 폼에 필수로 표시됩니다. 버튼 이름, 버튼 배경 색상, 버튼 텍스트 색상을 변경할 수 있습니다.

## 구독 확인 이메일 <a href="#confirm_email" id="confirm_email"></a>

구독 신청 시 발송되는 이메일입니다.

구독 확인 이메일을 사용하는 경우, 구독자는 이메일에 포함된 구독 확인 링크를 클릭해야 주소록에 구독자로 등록됩니다 24시간이 지났다면 다시 구독 신청을 진행해 새로운 확인 링크를 받아 주세요.

* 확인 링크 유효 시간: 24시간

#### 발송 여부 설정하기

\[구독 확인 이메일 발송하기]를 클릭하면 구독 확인 이메일 발송 여부가 변경됩니다.&#x20;

* 활성화: 구독 신청 시 주소록에 바로 구독자로 추가됩니다. 구독 과정이 단축되지만, 구독자가 잘못된 이메일 주소를 입력했을 때 확인이 어려울 수 있습니다.
* 비활성화: 구독 확인 이메일이 발송됩니다. 확인 과정이 추가되지만, 잘못된 이메일 주소가 등록되거나 스팸 봇에 의한 구독 신청을 방지할 수 있습니다.

{% hint style="info" %}
구독 신청 과정에서 잘못된 이메일 주소를 입력했거나 혹은 스팸성 이메일 발송을 위해 임의로 구독 신청을 진행하는 구독자가 존재할 수 있습니다. 안정적인 발송을 위해 되도록 구독 확인 이메일 기능은 사용해 주시는 걸 권장드립니다.

자세한 내용은 [뉴스레터에 구독 확인 이메일이 필요한 이유](https://blog.stibee.com/nyuseureteoe-gudog-hwagin-imeili-pilyohan-iyu/) 블로그 글을 확인해 주세요.
{% endhint %}

![](<../../.gitbook/assets/5 (8).png>)

#### &#x20;구독 확인 이메일 수정하기

\[구독 확인 이메일 → 수정하기]에서 내용을 수정할 수 있습니다.

* 발신자 이름, 발신자 이메일 주소: 기본값은 주소록의 발신자 이름과 발신자 이메일 주소입니다.
* 콘텐츠: 구독 확인 이메일의 본문 내용입니다.
* 버튼: 구독 확인 버튼의 이름, 버튼 배경 색상, 버튼 텍스트 색상을 변경할 수 있습니다.

\[미리보기]를 클릭하면 구독 확인 이메일이 어떻게 표시되는지 확인할 수 있습니다.

**\*주의:** 구독 확인 이메일에서 [메일머지](../../email/edit/personalized-merge.md) 기능이 동작하지 않습니다. 또한, 주소록 정보를 수정해도 구독 확인 이메일의 내용은 자동으로 변경되지 않으므로, 반드시 별도로 수정해 주세요.

## 내 홈페이지에 구독 폼 넣기 <a href="#adding-subscription-form" id="adding-subscription-form"></a>

{% hint style="info" %}
사용하는 홈페이지나 블로그의 특성에 따라 붙여 넣은 코드가 스티비에서 편집한 것과 다르게 보일 수 있어요. 이는 해당 서비스의 담당자나 제공 업체 측으로 문의해 주세요.
{% endhint %}

구독 폼을 HTML 코드 형태로 내보내 홈페이지나 블로그에 직접 삽입할 수 있습니다.

이 방법으로 원하는 위치에 원하는 형태로 \[구독 폼]을 노출할 수 있습니다. 만약, 코드를 직접 수정할 수 있다면 독립된 구독 페이지를 사용할 때보다 더 자유롭게 구성, 디자인을 수정할 수 있습니다.

1. \[주소록 → 구독 관리 화면 → 구독 폼]으로 이동합니다.
2. \[코드로 설치하기]를 클릭합니다.
3. \[모두 복사하기]를 클릭해 HTML 코드를 복사합니다.
4. 홈페이지 또는 블로그에 코드를 삽입합니다.

언어를 '영어'로 선택하면, 영어 버전의 HTML 코드도 사용할 수 있습니다. 단, 구독 폼 제목, 내용, 구독자 입력 항목, 버튼에 표시되는 문구 등은 \[수정하기]를 통해 직접 변경해 주셔야 합니다.

\[코드 내보내기]를 사용해 만들어진 구독 폼이 궁금하다면 아래 예시를 참고해 보세요.

* [스티비 블로그](https://blog.stibee.com/nurturing-email-for-sales-conversion)

#### 참고사항

구독 폼을 HTML 코드로 편집할 때 몇 가지 참고하실 점이 있습니다.

* 제공되는 것은 HTML 코드만이며, CSS 수정은 직접 진행해야 합니다.
* `id`, `class` 값을 수정하면 정상 작동하지 않을 수 있습니다.
* 하나의 페이지에는 하나의 구독 폼만 사용할 수 있습니다.
* 코드 내보내기로 생성한 구독 폼은 스티비 기능 업데이트가 자동 반영되지 않으므로, 기능 변경 시 코드를 다시 내보내야 합니다.
  * 마찬가지로 \[구독 폼] 내용을 수정했다면 코드를 다시 내보내기 해서 사용해야 합니다.
* 자동 입력 방지(reCAPTCHA)는 코드 내보내기 방식에서는 적용되지 않습니다.

![](<../../.gitbook/assets/8 (8).png>)



구독 폼뿐만 아니라 \[페이지]를 사용해도 구독 신청을 받을 수 있습니다. 페이지를 사용한 구독 방법은 [페이지로 구독자 모으기](../../page/subscribe/) 도움말을 참고해 주세요.
