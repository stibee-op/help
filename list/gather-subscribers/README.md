# 구독자 모으기

## 이 글에서는

구독자를 모을 수 있는 기능인 구독 폼에 대해서 알아봅니다.

***

스티비에서는 구독자를 모을 수 있는 기능으로 \[구독 폼]을 제공합니다. [구독 폼](form.md)뿐만 아니라 [페이지](broken-reference)를 통해서도 구독 신청을 받을 수 있지만, 이 섹션에서는 구독 폼을 활용해 구독자를 모으는 방법에 대해서 알아봅니다.

구독 폼에서 제공하는 기능을 활용하면 구독자를 미리 특정 그룹이나 세그먼트로 분류할 수 있습니다. 이 섹션에서는 구독 폼 사용 방법에 대해서 알아보세요.
