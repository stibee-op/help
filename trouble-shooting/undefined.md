# 서비스를 원활하게 사용할 수 없을 때

## 언제 필요한가요? <a href="#h_01hkm26fsyrr8kbfdtmncznw7h" id="h_01hkm26fsyrr8kbfdtmncznw7h"></a>

* 스티비 서비스 사용 중 문제가 생겼을 때
* 스티비 서비스 사용이 원활하지 않을 때
* 화면이 느리게 움직이거나 처리될 때

***

서비스를 사용하다 보면 기능이 원활하게 동작하지 않을 때가 있습니다. 아래 가이드를 참고하여 문제의 원인을 찾고 해결해 보세요.



### 발송(예약)을 진행했으나 처리되지 않을 때

\[발송하기] 또는 \[예약하기] 버튼을 눌러도 반응이 없거나 다음 단계로 넘어가지 않는다면 아래 순서대로 확인해 주세요.

#### 시크릿 모드에서 다시 시도하기

브라우저에 설치된 확장 프로그램이나 캐시 문제로 인해 버튼이 동작하지 않을 수 있습니다. 정확한 원인 확인을 위해 크롬 시크릿 모드에서 다시 한 번 작업을 시도해 주세요.

시크릿 모드는 아래 방법으로 실행할 수 있습니다.

* 윈도우: Ctrl + Shift + N
* 맥: ⌘ + Shift + N

시크릿 모드에서 정상적으로 동작한다면 확장 프로그램 또는 캐시 문제입니다. [시크릿 모드에서는 정상적으로 동작한다면](https://www.help.stibee.com/trouble-shooting/overview#works-in-incognito-mode) 도움말을 참고해 이후 조치를 진행해 주세요.

#### 스타일 제거하기

눈에 보이지 않는 불필요한 코드가 포함되어 버튼이 작동하지 않는 경우가 있습니다. 아래 방법으로 스타일 제거를 진행해 주세요.

1. 텍스트를 모두 선택한 뒤 \[툴바 → 포맷 초기화(T자에 \ 빗금이 있는 아이콘)] 버튼을 누릅니다.
2. 서식이 모두 초기화되면 다시 발송(예약)을 시도합니다.
3. 문제가 반복된다면, 텍스트 상자를 삭제한 뒤 '서식 없이 붙여넣기(Ctrl + Shift + V / ⌘ + Shift + V)'로 내용을 다시 입력해 주세요.

위 방법으로 문제가 해결됐다면, 외부에서 텍스트를 복사하는 과정에서 눈에 보이지 않는 불필요한 코드나 공백이 추가된 상황입니다. 구글 워드, 노션, 챗지피티 등 외부에서 복사한 텍스트에는 눈에 보이지 않는 여러 코드가 포함되어 있습니다.&#x20;

따라서, 외부에서 텍스트를 복사하실 때는 스타일 제거나 '서식 없이 붙여넣기(ctrl+shift+v)'를 활용해 주세요.



문제가 계속되는 경우에는 1:1 채팅 상담을 통해 문의해 주세요. 문의 남기실 때는 문제가 발생한 이메일의 제목을 함께 알려주시면 더욱 빠른 확인이 가능합니다.



### 화면 전환이 느리게 처리될 때

혹시 주소록을 여러 개 사용하고 계신가요? 서비스의 구조상, 주소록이 많을수록 화면 전환 속도가 느려질 수 있습니다.

스티비의 구독자는 \[주소록] 단위로 관리됩니다. 안정적인 사용과 원활한 구독자 관리를 위해 주소록은 소수로 관리하는 것을 권장드립니다.

아래 방법으로 주소록을 합칠 수 있습니다.

1. \[주소록 → 구독자 목록 → 내려받기 → 구독자 목록 내려받기]를 눌러, 각 주소록의 구독자 목록을 파일로 내려받습니다.
2. 새 주소록을 만든 뒤 \[구독자 추가하기 → 파일로 추가하기]에서 1번에서 내려받은 파일을 업로드합니다.

#### 시크릿 모드에서 다시 시도하기

주소록을 많이 생성하지 않았는데도 화면이 느리게 전환된다면, 브라우저 확장 프로그램이나 캐시 문제일 수 있습니다. 정확한 원인을 확인하기 위해  크롬 시크릿 모드에서 다시 한 번 작업을 시도해 주세요.

시크릿 모드는 아래 방법으로 실행할 수 있습니다.

* 윈도우: Ctrl + Shift + N
* 맥: ⌘ + Shift + N

시크릿 모드에서 정상적으로 동작한다면 확장 프로그램 또는 캐시 문제입니다. [시크릿 모드에서는 정상적으로 동작한다면](https://www.help.stibee.com/trouble-shooting/overview#works-in-incognito-mode) 도움말을 참고해 이후 조치를 진행해 주세요.



문제가 계속되는 경우에는 1:1 채팅 상담을 통해 문의해 주세요. 문의 남기실 때는 문제가 발생한 이메일의 제목을 함께 알려주시면 더욱 빠른 확인이 가능합니다.
