# 텍스트가 굵게 표시될 때

## 언제 필요한가요? <a href="#h_ea0d3fae83" id="h_ea0d3fae83"></a>

* 텍스트에 볼드 스타일을 적용하지 않았으나, 에디터나 이메일에서 글씨가 굵게 보일 때
* 다른 사람의 환경에서는 이메일이 정상적으로 표시되지만, 내 환경에서만 이메일 글씨가 굵게 보일 때

***

## 원인 살펴보기

이메일에서는 웹폰트를 사용할 수 없으므로 사용자(수신자)의 시스템에 설치된 폰트를 불러와서 표시합니다.

예를 들어, 스티비 에디터에서 폰트들 '고딕'으로 설정한 경우

* 애플고딕 → 노토산스 → 나눔고딕 → 맑은고딕 순으로 표시합니다.
* 애플고딕이 없다면 노토산스를, 노토산스가 없다면 나눔고딕으로 표시하는 방식입니다.

최근 설치된 프로그램이 폰트 시스템에 영향을 주었거나, 특정 폰트 파일이 손상된 경우에 발생합니다.



## 문제 해결하기

### 크롬 시크릿 모드에서 확인하기

시크릿 모드는 크롬 브라우저에서 아래 단축키를 눌러 실행할 수 있습니다.

* 윈도우: Ctrl + Shift + N
* 맥: ⌘ + Shift + N

시크릿 모드에서는 폰트가 정상적으로 표시된다면, 일반 모드에 설치된 '확장 프로그램'이 문제일 수 있습니다. 최근에 설치된 확장 프로그램을 하나씩 비활성화하여 원인이 되는 프로그램을 찾을 수 있습니다.

### 다른 이메일의 폰트 굵기 확인하기

크롬 시크릿 모드에서도 굵게 표시된다면, 이전에 정상적으로 표시됐던 이메일도 확인합니다. 다른 이메일은 정상적으로 표시된다면, 특정 이메일에서만 발생하는 문제일 수 있습니다.

아래 방법으로 포맷 초기화를 진행해 보세요.

1. 본문에 적힌 텍스트 전체를 드래그합니다.
2. 검정색 툴바에 있는 \[포맷초기화]를 클릭합니다. (T자에 \ 빗금이 있는 아이콘입니다.)   \
   &#xNAN;_\* 포맷 초기화는 텍스트의 스타일이 모두 초기화됩니다._

만약, 문제가 계속된다면 정확한 확인을 위해 [채팅 상담](../getting-started/undefined-4.md#help-center)으로 문의 부탁드립니다.

### 폰트 다시 설치하기

다른 이메일도 모두 굵게 표시되거나, 다른 사람의 환경에서는 정상이라면 현재 사용하고 계신 환경의 문제일 확률이 높습니다. 아래 방법으로 폰트 재설치를 권장드립니다.

{% hint style="info" %}
다음 가이드 내용은 '크롬 브라우저'를 기준으로 작성이 되었기 때문에 다른 브라우저에서는 확인이 어려울 수 있습니다.
{% endhint %}

1. 문제가 발생한 이메일 편집 단계(콘텐츠)에서 \[F12]를 눌러 '크롬 개발자 도구' 열기
2. 문제되는 텍스트를 클릭한 뒤 \[computed 탭 → Rendered Fonts] 확인
3. 표시된 폰트가 문제 원인이므로, 아래 도움말을 참고해 원인이 된 폰트를 삭제합니다.

* 윈도우: \[글꼴] 폴더에서 삭제 가능 [윈도우 환경에서 폰트 삭제하기](https://support.microsoft.com/ko-kr/office/%EC%82%AC%EC%9A%A9%ED%95%98%EC%A7%80-%EC%95%8A%EB%8A%94-%EC%96%B8%EC%96%B4-%EB%B0%8F-%EA%B8%80%EA%BC%B4-%EC%A0%9C%EA%B1%B0-0915efdf-3eec-4e87-8b2f-ba0d47242ea6)
* 맥: \[서체 관리자]에서 삭제 가능 [Mac 환경에서 폰트 삭제하기](https://support.apple.com/ko-kr/guide/font-book/fntbk1000/mac)

4. 컴퓨터를 재부팅해 폰트가 정상적으로 표시되는지 확인

간혹 폰트를 삭제해도 문제가 계속되는 경우가 있습니다. 이 경우 문제의 원인이 된 폰트와 같은 계열의 폰트를 모두 삭제한 뒤에 다시 재설치하는 방법이 있습니다.&#x20;

예를 들어, Noto Sans CJK KR Black 폰트가 문제라면 이 폰트와 같은 계열의 모든 Noto Sans 폰트를 삭제 후 재설치하는 것이 좋습니다. 폰트는 [폰트 다운로드 받기](https://github.com/googlefonts/noto-cjk/raw/main/Sans/Variable/TTF/NotoSansCJKkr-VF.ttf) 링크에서 다운로드 할 수 있습니다.



문제가 계속되는 경우에는 1:1 채팅 상담을 통해 문의해 주세요. 문의 남기실 때는 문제가 발생한 이메일의 제목을 함께 알려주시면 더욱 빠른 확인이 가능합니다.
