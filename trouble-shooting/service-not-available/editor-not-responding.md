# 이메일을 편집할 수 없을 때

## 언제 필요한가요? <a href="#h_01hkm26fsyrr8kbfdtmncznw7h" id="h_01hkm26fsyrr8kbfdtmncznw7h"></a>

* 이메일 편집 기능이 정상적으로 동작하지 않을 때

***

이메일을 편집하기 위해 에디터 화면을 눌러도 반응이 없다면 아래 순서대로 확인해 주세요.

### 시크릿 모드에서 다시 시도하기 <a href="#incognito-mode" id="incognito-mode"></a>

브라우저에 설치된 확장 프로그램이나 캐시 문제로 인해 버튼이 동작하지 않을 수 있습니다. 정확한 원인 확인을 위해 크롬 시크릿 모드에서 다시 한 번 작업을 시도해 주세요.

아래 방법으로 실행할 수 있습니다.

* 윈도우: Ctrl + Shift + N
* 맥: ⌘ + Shift + N

시크릿 모드에서 정상적으로 동작한다면 확장 프로그램 또는 캐시 문제입니다. [시크릿 모드에서는 정상적으로 동작한다면](https://www.help.stibee.com/trouble-shooting/overview#works-in-incognito-mode) 도움말을 참고해 이후 조치를 진행해 주세요.



### 스타일 제거하기 <a href="#delete-style" id="delete-style"></a>

크롬 시크릿 모드에서도 정상적으로 동작하지 않는다면, 눈에 보이지 않는 불필요한 코드가 포함되어 편집 화면이 정상적으로 동작하지 않는 경우일 수 있습니다.&#x20;

아래 방법으로 스타일 제거를 진행해 주세요.

1. 텍스트를 모두 선택한 뒤 \[툴바 → 포맷 초기화(T자에 \ 빗금이 있는 아이콘)] 버튼을 누릅니다.
2. 서식이 모두 초기화되면 다시 발송(예약)을 시도합니다.
3. 문제가 반복된다면, 텍스트 상자를 삭제한 뒤 '서식 없이 붙여넣기(Ctrl + Shift + V / ⌘ + Shift + V)'로 내용을 다시 입력해 주세요.

위 방법으로 문제가 해결됐다면, 외부에서 텍스트를 복사하는 과정에서 눈에 보이지 않는 불필요한 코드나 공백이 추가된 상황입니다. 구글 워드, 노션, 챗지피티 등 외부에서 복사한 텍스트에는 눈에 보이지 않는 여러 코드가 포함되어 있습니다.&#x20;

따라서, 외부에서 텍스트를 복사하실 때는 스타일 제거나 '서식 없이 붙여넣기(ctrl+shift+v)'를 활용해 주세요.



위 단계를 모두 시도했는데도 문제가 해결되지 않았다면, 1:1 채팅 상담을 통해 문의해 주세요. 문의 시, 문제가 발생한 이메일 제목을 함께 알려주시면 빠르게 확인해 드릴 수 있습니다.
