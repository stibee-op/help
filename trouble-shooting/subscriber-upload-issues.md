# 구독자 파일 업로드가 안 될 때

## 언제 필요한가요? <a href="#h_01he4y5t97t3c2qjd5nbbww9qr" id="h_01he4y5t97t3c2qjd5nbbww9qr"></a>

* 구독자를 파일로 업로드했으나 오류가 발생할 때
* 구독자를 파일로 업로드했지만 \[다음] 버튼이 활성화되지 않아 진행이 어려울 때
* 구독자 정보를 선택하고 업로드를 눌렀지만, 오류가 발생하며 진행이 되지 않을 때
* 파일을 업로드했지만, 파일의 숫자와 실제 업로드된 구독자의 수가 다를 때

***

업로드한 파일의 형식이 올바르지 않거나, 브라우저, 네트워크 환경 등의 문제로 파일이 원활하게 업로드되지 않을 수 있습니다. 아래 순서에 따라 어떤 부분이 잘못됐는지 단계적으로 확인해 보세요.

### 파일 형식 확인하기

파일의 형식이 CSV인가요? CSV 형식이 아닌 다른 확장자는 업로드할 수 없습니다. 또한 확장자가 CSV더라도 저장 과정에서 파일 형식 오류가 생겼을 수 있습니다.&#x20;

아래 방법으로 파일을 다시 저장해 보세요.

#### 엑셀에서 저장하기

{% hint style="info" %}
엑셀 버전에 따라 메뉴 이름이 다를 수 있습니다.
{% endhint %}

1. 엑셀을 실행한 뒤 \[파일 열기]로 업로드할 파일을 불러옵니다.
2. \[다른 이름으로 저장하기]를 클릭합니다.
3. 파일 형식을 'CSV UTF-8(쉼표로 분리) (\*.csv)'로 선택해 저장합니다.

<figure><img src="../.gitbook/assets/image (96).png" alt=""><figcaption></figcaption></figure>



#### 메모장(윈도우 전용)에서 저장하기 <a href="#undefined" id="undefined"></a>

1. 메모장을 실행한 뒤 \[파일 열기]를 클릭합니다.
2. 파일 형식을 \[모든 파일 (.)] 로 변경하고 CSV 파일을 불러옵니다.
3. \[파일 → 다른 이름으로 저장]을 누르고, 인코딩을 'UTF-8'로 선택해 저장합니다.

<figure><img src="../.gitbook/assets/image (97).png" alt=""><figcaption></figcaption></figure>



### 구독자 정보 형식 확인하기

원본 파일에 잘못된 형식의 정보가 포함되어 있으면 업로드가 실패할 수 있습니다. 아래 기본 형식을 확인하거나 샘플 파일을 내려받아 비교해 보세요.

* &#x20;[샘플 파일 내려받기](https://stibee.com/download/%EC%8A%A4%ED%8B%B0%EB%B9%84_%EC%A3%BC%EC%86%8C%EB%A1%9D_%EC%83%98%ED%94%8C.csv)

#### 기본 입력 형식

* 구독자 이메일 주소가 한 열(컬럼)에 입력되어야 합니다.&#x20;
* 제목 행(이메일 주소, 이름)은 없어도 됩니다.
* 이메일 주소 외의 열은 필수가 아닙니다.
* 파일 업로드 후 각 열을 어떤 필드에 매칭할지 직접 선택할 수 있으므로, 열 순서나 제목은 크게 상관없습니다.

#### 날짜, 시간 타입 형식 <a href="#h_01hehqt3qngp7fbhy4zfm55196" id="h_01hehqt3qngp7fbhy4zfm55196"></a>

날짜나 시간 정보를 추가했다면 지원하는 형식으로 입력되어 있는지 확인해 주세요. 지원하지 않는 형식은 업로드 과정에서 오류를 일으킬 수 있습니다.

**\*주의:** 시간은 24시간 기준으로 입력해야 합니다.

* 예를 들어, 2023년 9월 20일 오후 5시 25분 15초를 추가하고 싶다면, 2023-09-20 17:25:15 형식으로 입력해야 합니다.

1995년 5월 6일이 생일인 구독자가 있습니다. 생일을 주소록에 추가하고 싶습니다.

* 1995-05-06 형식으로 입력하면 생일이 추가됩니다.
* 1995-5-6 형식으로 입력하면 생일이 추가되지 않습니다.

2023년 11월 24일 오후 10시 35분 15초에 구독한 구독자가 있습니다. 구독일과 구독한 시간을 추가하고 싶습니다.

* 2023.11.23 22:35:15 형식으로 입력하면 구독일과 시간이 추가됩니다.
* 2023.11.23 오후 10:35:15 형식으로 입력하면 오전 10시 35분으로 추가됩니다. 시간은 24시간 형식으로 입력해야 합니다.



### 사용자 정의 필드 올바르게 매칭하기 <a href="#h_01h0s8dz775dyr4awtwwed30xp" id="h_01h0s8dz775dyr4awtwwed30xp"></a>

파일 업로드 후, 이메일 주소를 제외한 나머지 필드는 기본적으로 \[사용 안 함] 으로 설정됩니다. 주소록에 추가해야 하는 정보가 있다면 원본 파일의 열과 사용자 정의 필드를 정확히 매칭해 주세요.

### 파일 용량 확인하기

업로드 가능한 최대 파일 용량은 50MB입니다. 이메일 주소만 입력된 경우 약 250만 행까지 업로드할 수 있습니다. 더 큰 용량의 파일이라면 50MB 이하로 나누어 업로드하는 것을 권장드립니다.



### 시크릿 모드에서 확인하기

브라우저 확장 프로그램이나 캐시 문제로 인해 정상적인 파일 업로드가 어려울 수 있습니다. 크롬 시크릿 모드로 전환한 뒤 다시 업로드를 시도해 보세요.

크롬 시크릿 모드는 아래 방법으로 실행할 수 있습니다.

* 윈도우: Ctrl + Shift + N
* 맥: ⌘ + Shift + N

시크릿 모드에서는 정상적으로 업로드된다면 브라우저 확장 프로그램이나 캐시가 원인일 가능성이 높습니다. 캐시를 삭제하시거나 확장 프로그램을 하나씩 비활성화하며 원인을 확인해 주세요.



### 다른 네트워크에서 업로드하기

네트워크 보안 설정으로 인해 파일 업로드가 제한될 수 있습니다. 현재 네트워크가 아닌 다른 네트워크(예. 모바일 테더링 등) 에서 업로드를 시도해 보세요.

다른 네트워크에서는 정상적으로 업로드된다면 보안 설정 문제일 수 있으며, 이 경우 내부 보안 관리자에게 문의하면 빠르게 해결할 수 있습니다.
