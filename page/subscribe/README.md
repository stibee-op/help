# 페이지로 구독자 모으기

## 이 글에서는

페이지에서 구독자를 모집하는 방법에 관해 알아봅니다.

***

페이지에서도 구독자를 모을 수 있습니다. 페이지뿐만 아니라 [구독 폼](../../list/gather-subscribers/form.md)에서도 구독 신청을 받을 수 있지만, 이 섹션에서는 페이지를 활용해 구독자를 모으는 방법에 대해서 알아봅니다.

페이지에서 제공하는 기능을 활용하면 구독자를 미리 특정 그룹이나 세그먼트로 분류할 수 있습니다.
