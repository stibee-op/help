# 이미지

## 이 글에서는

에디터에서 이미지를 추가하고, 크기와 모양을 조절하거나 이미지에 링크를 넣는 방법을 설명합니다.

***

### 이미지 업로드하기 <a href="#h_01ggs6kwt85pwrr5tka6f34mgr" id="h_01ggs6kwt85pwrr5tka6f34mgr"></a>

오른쪽 구성 요소 화면에서 \[이미지] 상자를 클릭해 본문으로 끌어 놓거나, 구성 요소 목록에서 '이미지'를 선택해 이미지 상자를 추가합니다. 이미지 상자를 추가한 뒤 상자를 클릭하면 이미지를 업로드할 수 있는 창이 나타납니다.

_\* 이미지 파일은 jpg, png, gif 형식을 지원합니다._



### 이미지 용량 제한

요금제에 따라 이미지 제한 용량이 달라집니다.&#x20;

* 스타터 요금제: 5mb 제한
* 스탠다드, 프로, 엔터프라이즈 요금제 : 용량 제한 없음

_\* 이미지 파일은 jpg, png, gif 형식을 지원하며, 원본 그대로 업로드됩니다._

**\*주의:** 이미지 1장당 용량은 요금제와 관계없이 5MB로 제한합니다.



### 이미지 정렬, 링크, 모서리, 대체 텍스트 설정하기

이미지가 삽입된 상자를 클릭하면 상자 위에 이미지 툴바가 표시됩니다. 툴바에서 다음 항목을 설정할 수 있습니다.

* 정렬: 왼쪽, 오른쪽, 가운데, 양쪽 중에서 선택합니다.
*   링크: \[링크(🔗)]를 클릭해 URL을 추가하거나 삭제합니다.

    _\* 최대 999자의 URL을 추가할 수 있습니다._
* 모서리 둥글게: \[모서리 둥글게]를 클릭하면 이미지의 네 모서리가 둥글게 처리됩니다.
* 대체 텍스트: 툴바 가장 오른쪽의 \[Alt]를 클릭해 대체 텍스트를 입력합니다.

### 이미지 편집하기 <a href="#h_01ggs6m5jy3tvtsvjtp8nxe66e" id="h_01ggs6m5jy3tvtsvjtp8nxe66e"></a>

이미지 툴바에서 \[이미지 편집] 버튼을 클릭하면 이미지 편집 화면으로 이동합니다. 이미지 자르기, 회전, 좌우 반전 등 간단한 이미지 편집을 할 수 있습니다.

### 이미지 크기 변경하기 <a href="#h_01ggs6m9zsvgrh7smtf4ehfaen" id="h_01ggs6m9zsvgrh7smtf4ehfaen"></a>

\[이미지 편집] 버튼 옆의 \[이미지 크기 조절] 버튼을 클릭하면 이미지 크기를 조절할 수 있는 슬라이드가 표시됩니다. 슬라이드를 좌우로 움직여 픽셀(px) 단위로 크기를 조절합니다.
